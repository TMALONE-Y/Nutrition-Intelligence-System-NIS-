## 📌 Overview

**Nutrition Intelligence System (NIS)** is a Python-based CLI application designed to calculate daily calorie needs and macronutrient distribution based on user-specific data and fitness goals.

This project applies real-world health formulas and data-driven logic to help users understand exactly how much they should eat to either build muscle or lose fat.

---

## 🎯 Features

- 🔹 BMR (Basal Metabolic Rate) calculation  
- 🔹 Maintenance calorie estimation based on activity level  
- 🔹 Goal selection:
  - Muscle Building (Bulking)
  - Fat Loss (Cutting)  
- 🔹 Macronutrient breakdown:
  - Protein (grams + %)
  - Carbohydrates (grams + %)
  - Fats (grams + %)  
- 🔹 Input validation & error handling  
- 🔹 Clean CLI user experience  
- 🔹 Personalized fitness tips  

---

## 🧠 How It Works

1. User enters:
   - Age  
   - Height  
   - Weight  
   - Activity level  

2. System calculates:
   - BMR  
   - Maintenance Calories  

3. Based on goal:
   - Adds calorie surplus (Bulking)  
   - Applies calorie deficit (Cutting)  

4. Outputs:
   - Daily calories  
   - Protein, carbs, fats (grams + percentages)
